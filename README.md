
# Meghna - Plantilla Gratuita de Bootstrap

Meghna es una plantilla de Bootstrap gratuita y receptiva, adecuada para sitios web de negocios, agencias y portafolios.

## Sobre Meghna

Meghna es una plantilla limpia y moderna construida con Bootstrap, HTML5, CSS3 y jQuery. Es totalmente receptiva y se ve muy bien en todos los dispositivos.

## Características

*   Totalmente receptiva
*   Bootstrap 4
*   HTML5 y CSS3
*   JQuery
*   Soporte para Gulp
*   Soporte para SASS
*   Diseño limpio y moderno
*   Compatible con todos los navegadores

## Instalación

1.  Clona el repositorio:
    ```
    git clone https://github.com/themefisher/meghna-bootstrap.git
    ```
2.  Navega al directorio del proyecto:
    ```
    cd meghna-bootstrap
    ```
3.  Instala las dependencias:
    ```
    npm install
    ```

## Ejecutando el proyecto

Para ejecutar el proyecto en modo de desarrollo, usa el siguiente comando:

```
npm run dev
```

Esto iniciará un servidor local y abrirá el proyecto en tu navegador. El navegador se recargará automáticamente cuando realices cambios en los archivos de origen.

Para construir el proyecto para producción, usa el siguiente comando:

```
npm run build
```

Esto creará un directorio `theme` con todos los archivos necesarios para el despliegue.

## Estructura de Carpetas

```
meghna-bootstrap/
├── source/
│   ├── html/
│   ├── scss/
│   ├── js/
│   ├── images/
│   ├── plugins/
│   └── partials/
├── theme/
│   ├── css/
│   ├── js/
│   ├── images/
│   └── plugins/
├── gulpfile.js
├── package.json
└── README.md
```

*   **source/**: Contiene los archivos de origen del proyecto.
    *   **html/**: Contiene los archivos HTML.
    *   **scss/**: Contiene los archivos SASS.
    *   **js/**: Contiene los archivos JavaScript.
    *   **images/**: Contiene las imágenes.
    *   **plugins/**: Contiene los plugins.
    *   **partials/**: Contiene los parciales para los archivos HTML.
*   **theme/**: Contiene los archivos distribuibles del proyecto.
*   **gulpfile.js**: Contiene las tareas de Gulp.
*   **package.json**: Contiene las dependencias y scripts del proyecto.
*   **README.md**: Contiene la documentación del proyecto.
